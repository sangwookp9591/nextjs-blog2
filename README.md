# QUICK MOTION NFT FRONTEND

Quick Motion NFT frontend Web Service with React, NextJs

## Prerequisites

- [Node.js > 10](https://nodejs.org) and npm

## Getting Started

1. Install Dependencies (npm or yarn)

```bash
- yarn : `yarn install`
- npm : `npm install`
```

2. RUN - DEV

```bash
- yarn : `yarn run dev`
- npm : `npm run dev`
```

3. RUN - PRODUCTION

```bash
- yarn : `yarn run dev`
- npm : `npm run dev`
```

4. RUN - BUILD

```bash
- yarn : `yarn run dev`
- npm : `npm run dev`
```
